# vanilla-js-calculator

## Description
A responsive calculator built using HTML, CSS, and Vanilla JavaScript with syntax validation and dynamic DOM manipulation.

## Features
- Dynamic DOM manipulation
- Syntax validation
- Delete and Clear functionality
- Error handling
- Real-time button interaction

## Tech Used
- HTML
- CSS
- JavaScript
